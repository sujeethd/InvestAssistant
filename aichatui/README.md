# aichatui

Claude-inspired React UI for the InvestAssistant hybrid chat flow.

## Run locally

```bash
cd aichatui
npm install
npm run dev
```

The UI is preconfigured to use the hosted chat service.
For theme changes, edit `aichatui/src/config.js`.

## Theme

Use the Theme selector to switch color palettes. Themes are defined in
`aichatui/src/styles.css` via CSS variables.
